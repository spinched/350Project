# Grocery Store Database
This repository contains the SQL files pertaining to this database to serve as a backup of the initial schema SQL files that build the database.
It also contains the front end files that make up the website used to view and interact with the database.
The full documentation for this project can be reached at this link: https://docs.google.com/document/d/1LRiwduMcD45Z8WUYqzmOFj_d7Z5VE55E/edit?usp=sharing&ouid=109786925889650846748&rtpof=true&sd=true
